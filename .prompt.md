### Phase 8: DESeq2 Differential Expression (Severity-Specific)

| Step | Source | What to Do |
|------|--------|------------|
| Set reference level | HW2 Step 4 | `metadata$severity_group <- relevel(factor(metadata$severity_group), ref = "Healthy_Mild")` |
| Create DESeqDataSet | HW2 Step 5 / Tutorial 4 | `dds <- DESeqDataSetFromMatrix(countData = counts, colData = metadata, design = ~ severity_group)` |
| Library size barplot | Tutorial 4 Phase 5 | `barplot(colSums(counts(dds)))` |
| Estimate size factors | Tutorial 4 Phase 5 | `dds <- estimateSizeFactors(dds); dds$sizeFactor` |
| Mean-variance plot | Tutorial 4 Phase 5 | Verify overdispersion: variance > mean for most genes |
| Run DESeq2 | HW2 Step 5 / Tutorial 4 | `dds <- DESeq(dds)` |
| Extract results | HW2 Step 5 / Tutorial 4 | `res <- results(dds)` |
| Shrink fold changes | HW2 Step 5 / Tutorial 4 | `resLFC <- lfcShrink(dds, coef = "severity_group_Severe_ICU_vs_Healthy_Mild", type = "apeglm")` |
| MA plot before/after | Tutorial 4 Phase 7 | `plotMA(res)` then `plotMA(resLFC)` |
| Map gene symbols | Tutorial 4 Phase 8 | `resLFC$symbol <- mapIds(org.Hs.eg.db, keys=rownames(resLFC), column="SYMBOL", keytype="ENSEMBL")` |
| Order by significance | Tutorial 4 Phase 9 | `resOrdered <- resLFC[order(resLFC$pvalue),]` |
| Count significant genes | Tutorial 4 Phase 6 | `sum(resLFC$padj < 0.05, na.rm=TRUE)` |
| Save results | Tutorial 4 Phase 9 | `write.csv(as.data.frame(resOrdered), "SevereICU_vs_HealthyMild_DESeq2.csv")` |

> **Decision Checkpoint 4:** Why DESeq2 for count data over edgeR or limma? Why group Severe+ICU together vs treating them separately?
